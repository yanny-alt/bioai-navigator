## Semantic Search Flow

1. **One-time**: Python script reads tool descriptions вҶ’ generates embeddings using `sentence-transformers` (all-MiniLM-L6-v2) вҶ’ saves as `embeddings.rds`
2. **Runtime**: User types query вҶ’ R computes cosine similarity between query embedding and stored embeddings вҶ’ returns ranked results
3. **Fallback**: If embeddings unavailable, falls back to keyword search via `stringr`

## Module Structure

Each tab is a self-contained Shiny module (UI + server) for maintainability and reusability.

## Data Flow

Google Sheets вҶ’ `googlesheets4::read_sheet()` вҶ’ reactive data frame вҶ’ modules

Ratings/submissions вҶ’ Google Sheets append via `googlesheets4::sheet_append()`
