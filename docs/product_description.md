# BioAI Navigator — Product Description

## What is it?

BioAI Navigator is a community-driven web platform for discovering, comparing, and evaluating AI-powered tools in biological research. It serves as a structured, searchable directory — think of it as **Product Hunt for AI tools in biology**.

## Problem it solves

AI tools for biology are emerging rapidly across genomics, protein design, drug discovery, microscopy, and literature mining. Researchers struggle to discover, compare, and evaluate these tools because information is scattered across papers, GitHub repositories, Twitter/X threads, and preprint servers.

BioAI Navigator centralizes this knowledge in one searchable, filterable, community-maintained platform.

## Key differentiators

1. **Semantic search** — Goes beyond keyword matching. Uses sentence embeddings (all-MiniLM-L6-v2) to understand the *intent* of a query and surface contextually relevant tools.

2. **Curated metadata** — Every tool entry is human-curated with standardized fields: domain, AI technique, data type, publication, license, tags.

3. **Community layer** — Researchers can rate, review, and submit tools, making it a living directory rather than a static list.

4. **Side-by-side comparison** — Users can compare 2–3 tools on all metadata dimensions at once.

## Target Users

- Computational biologists and bioinformaticians looking for the right tool
- Wet lab scientists exploring AI for their workflows
- Graduate students entering the field
- Educators and instructors
- AI researchers building new biology tools

## Design inspiration

- Product Hunt (card-based tool listings)
- PapersWithCode (method-centric, linked to papers)
- HuggingFace Model Hub (community ratings, rich metadata)
- bio.tools (structured biological tool registry)
